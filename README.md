# ACD GMP Toulon 2026

Site événementiel de l'Assemblée des Chefs de Département GMP, organisée à l'IUT de Toulon les 3, 4 et 5 juin 2026.

## Aperçu

Site one-page statique. Tout le contenu, les styles et les scripts tiennent dans `index.html`.

## Lancer en local

Ouvrir `index.html` dans un navigateur. Aucun serveur n'est nécessaire.

## Déploiement

Le site est déployé via GitHub Pages depuis la branche `main` (racine).

## Structure

```
.
├── index.html        # site complet (HTML + CSS + JS)
├── assets/           # images : logos, plans de lignes de bus, photo de la rade
└── README.md
```
